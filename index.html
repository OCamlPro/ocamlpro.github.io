OCaml Tooling

  • opam: OCaml Package Manager
    • Sources: https://github.com/ocaml/opam
  • opam-bin: create and share binary packages for opam
    • Sources: https://github.com/ocamlpro/opam-bin
    • Relocation patches: https://github.com/ocamlpro/relocation-patches
  • drom: create OCaml projects in one command, easy interface on dune/opam
    • Sources: https://github.com/ocamlpro/drom
  • ocp-index: project navigator
    • Sources: https://github.com/ocamlpro/ocp-index
  • ocaml-top: A simple cross-platform OCaml code editor built for top-level evaluation.
    • Sources: https://github.com/ocamlpro/ocaml-top
  • tryocaml: An interactive tutorial to begin learning with OCaml
    • Sources: https://github.com/ocamlpro/tryocaml
  • ocp-indent: code indentation for editors
    • Sources: https://github.com/ocamlpro/ocp-indent
  • operf-micro: micro-benchmarking tool for the Ocaml compiler
    • Sources: https://github.com/ocamlpro/operf-micro
  • operf-macro: Some macro-benchmarks for operf and an OPAM repository for them
    • Sources: https://github.com/ocamlpro/operf-macro
  • ocp-build: composable build manager (deprecated)
    • Sources: https://github.com/ocamlpro/ocp-build
  • digodoc: generate online documentation for opam switches
    • Sources: https://github.com/ocamlpro/digodoc

OCaml Libraries (stable)

  • ocplib-endian: optimized byte-level manipulations
    • Sources: https://github.com/ocamlpro/ocplib-endian
  • ocp-reveal: binding for Reveal.JS presentation framework
    • Sources: https://github.com/ocamlpro/ocp-reveal
  • ez_cmdliner: simple interface à la Arg over Cmdliner
    • Sources: https://github.com/ocamlpro/ez_cmdliner
  • ez_subst: Easy substititions using ${X}, $(x), $[x], $x notations
    • Sources: https://github.com/ocamlpro/ez_subst
  • ez_opam_file: Compatibility layer over multiple versions of opam-file-format
    • Sources: https://github.com/ocamlpro/ez_opam_file

OCaml Libraries (unstable)

We use these libraries internally, so they tend to move a lot from time to time.

  • ez-api: Simplified Interface over ocplib-resto for Rest APIs
    • Sources: https://github.com/ocamlpro/ez-api
  • ez-pgocaml: Simplified Interface over pgocaml with db schema versioning for migrations
    • Sources: https://github.com/ocamlpro/ez-pgocaml
  • ez-cmdliner: Simplified Interface for Cmdliner with Arg compatibility
    • Sources: https://github.com/ocamlpro/ez-cmdliner

Languages and Formal Methods

  • alt-ergo: SMT Solver for program verification, initially developed at LRI, now maintained at OCamlPro
    • Sources: https://github.com/ocamlpro/alt-ergo
  • optal: A new language for optimization
    • Sources: https://github.com/ocamlpro/optal
  • scilint: Syntax analyzer for Scilab
    • Sources: https://github.com/ocamlpro/scilint

Blockchain

  • liquidity: Smart Contract Language compiling to Love (Dune) and Michelson (Tezos), including a decompiler to audit Michelson programs
    • Sources: https://github.com/ocamlpro/liquidity
  • tzscan: Open-source Block-explorer for Tezos
    • Sources: https://github.com/ocamlpro/tzscan
  • techelson: A test execution engine for Michelson smart contracts.
    • Sources: https://github.com/ocamlpro/techelson
